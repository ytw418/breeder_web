# Designer Assets

디자인 산출물 전용 폴더입니다. 개발 코드와 분리해 사용합니다.

## Structure
- `icons/`: 앱/브랜드 아이콘
- `banners/`: 랜딩/캠페인 배너
- `campaigns/`: 퍼포먼스 광고 크리에이티브
- `badges/`: 신뢰/인증 배지
- `posters/`: 포스터형 비주얼
- `og/`: Open Graph 공유 이미지

## Recommended Usage
- `campaigns/bredy-growth-ad-a`: 통합 가치 제안(탐색 + 거래 + 기록) 광고
- `campaigns/bredy-growth-ad-b`: 경매 신뢰 메시지 광고
- `campaigns/bredy-growth-ad-c`: 브리디북 기록 인증 광고
- `og/bredy-og-auction`: 경매 관련 URL 공유용 OG
- `og/bredy-og-bredybook`: 브리디북 관련 URL 공유용 OG
- `badges/bredy-trust-badge`: 인증 뱃지, 이벤트 보상, 프로필 스탬프
