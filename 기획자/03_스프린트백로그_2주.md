# 2주 스프린트 백로그

기간: 2026-02-12 ~ 2026-02-25  
목표: 검색/카테고리/알림 전환 지표 개선을 위한 릴리즈 1회 완료

## 우선순위 규칙
- P0: 이번 스프린트 내 반드시 완료
- P1: P0 완료 후 착수
- P2: 다음 스프린트 후보

## 백로그

1. `P0` 카테고리 값 정규화 정책 확정
- 설명: 상품/게시글에서 사용하는 카테고리 딕셔너리 고정
- 산출물: 카테고리 기준표 + API 파라미터 정의
- DoD: 제품/개발/운영이 같은 카테고리 용어를 사용

2. `P0` 상품 목록 필터 조합 테스트 케이스 작성
- 설명: `category`, `productType`, `status` 조합별 정상 동작 검증
- 산출물: QA 시나리오 문서 + 자동/수동 체크리스트
- DoD: 필터 조합 주요 케이스 회귀 테스트 통과

3. `P0` 통합 검색 정확도 개선 1차
- 설명: 상품명/설명/카테고리, 게시글 제목/내용, 사용자명 탐색 정합성 점검
- 산출물: 검색 품질 기준표(무관결과 정의 포함)
- DoD: 샘플 쿼리 30개 기준 무관 결과 비율 목표치 달성

4. `P0` 검색 무결과 화면 개선
- 설명: 무결과 시 재탐색 유도 UI 제공
- 산출물: 추천 키워드/카테고리 링크
- DoD: 무결과 후 이탈률 감소 추세 확인

5. `P0` 알림 unreadCount 정합성 점검
- 설명: 읽음 처리 API와 리스트 데이터 동기화 검증
- 산출물: 정합성 체크 로그 + 재현 시나리오
- DoD: 불일치 재현 케이스 0건

6. `P0` 알림 딥링크 매핑 표준화
- 설명: 타입별 목표 페이지 이동 규칙 문서화
- 산출물: 알림 타입-딥링크 매핑 표
- DoD: 타입별 샘플 클릭 테스트 모두 성공

7. `P1` 거래 상태 UX 카피/버튼 정리
- 설명: 판매중/예약중/판매완료 상태 전환 이해도 개선
- 산출물: 상태 전환 UX 문안 가이드
- DoD: 유저 테스트(내부 5명)에서 혼동 응답 20% 미만

8. `P1` 리뷰 작성 진입 개선
- 설명: 거래 완료 후 리뷰 작성 CTA 강화
- 산출물: 리뷰 유도 플로우 와이어
- DoD: 리뷰 작성 시작률 상승

9. `P2` 검색어 자동완성 PoC
- 설명: 인기 검색어/카테고리 후보 제안
- 산출물: 후보 노출 정책
- DoD: 기술 검증 완료 및 차기 스프린트 반영 여부 결정

## 데일리 운영
- 월/수/금: KPI 스냅샷 공유(등록, 검색, 알림, 채팅 시작)
- 화/목: 이슈 triage 및 우선순위 재정렬

## 스프린트 종료 체크
- 릴리즈 노트 작성
- KPI 전후 비교(최소 7일 관찰)
- 다음 스프린트로 이월할 항목 확정
