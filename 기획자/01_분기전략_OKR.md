# 2026 Q1 분기 전략 (OKR)

작성일: 2026-02-11  
기간: 2026-02-12 ~ 2026-03-31

## 북극성 지표
- `주간 거래 활성 사용자 수 (WTAU)`: 주간 기준 상품 등록/문의/거래완료 중 1회 이상 수행한 유저 수

## Objective 1. 거래 성공률을 높여 마켓 체감 가치를 만든다
- KR1. 거래 완료율(완료/등록)을 현재 대비 +25%
- KR2. 상품 상세 -> 채팅 시작 전환율 +20%
- KR3. 판매중 상태에서 14일 내 첫 문의 도달 비율 +30%

핵심 이니셔티브:
- 카테고리 체계 정비 및 필터 일관화
- 상품 카드/상세의 신뢰 정보(리뷰, 응답성, 상태) 강화
- 거래 상태 전환 UX 단순화(판매중/예약중/완료)

## Objective 2. 검색과 탐색 품질을 개선해 발견 비용을 낮춘다
- KR1. 검색 무결과 비율 -20%
- KR2. 검색 후 1분 내 클릭 발생률 +25%
- KR3. 검색 재시도 횟수(세션 평균) -15%

핵심 이니셔티브:
- 통합 검색 고도화(상품/게시글/유저 + 카테고리/상태 반영)
- 검색어 자동완성 후보(카테고리/인기 키워드) 도입
- 결과 정렬 기준 명시(최신순/인기순)

## Objective 3. 알림 신뢰도를 높여 재방문 루프를 만든다
- KR1. 알림 클릭률 +20%
- KR2. 알림 누락/중복 제보 건수 -50%
- KR3. 읽지 않은 알림의 24시간 내 소비율 +30%

핵심 이니셔티브:
- 알림 타입별 템플릿/라우팅 표준화
- 중복 발송 방지 로직 점검
- 알림 센터에서 컨텍스트 이동(대상 페이지 deep link) 안정화

## 측정 정의
- 거래 완료율 = `판매완료 상태 상품 수 / 신규 등록 상품 수`
- 검색 무결과 비율 = `검색 요청 중 결과 0건 비중`
- 알림 클릭률 = `알림 클릭 수 / 노출 알림 수`

## 운영 원칙
- 목표는 기능 수가 아니라 지표 변화로 판단
- KR 달성 실패 시 기능 추가보다 원인 분해 우선
- 신규 기능은 최소 1개 이상의 이벤트 로깅 기준을 먼저 정의
