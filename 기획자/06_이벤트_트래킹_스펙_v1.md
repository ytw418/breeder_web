# 이벤트 트래킹 스펙 v1

작성일: 2026-02-11  
목적: OKR/KPI 측정을 위해 최소 이벤트 집합을 표준화

## 1. 공통 규칙
- 이벤트명은 `snake_case` 사용
- 필수 파라미터: `user_id`, `timestamp`, `screen`, `source`
- 개인정보(전화번호/이메일/실명)는 이벤트 파라미터에 직접 저장 금지

## 2. 거래 퍼널 이벤트
- `product_list_viewed`
  - params: `category`, `product_type`, `status`, `sort`
- `product_detail_viewed`
  - params: `product_id`, `category`, `status`
- `chat_started_from_product`
  - params: `product_id`, `seller_id`
- `product_status_changed`
  - params: `product_id`, `from_status`, `to_status`

## 3. 검색 이벤트
- `search_executed`
  - params: `query`, `search_type`, `result_count`, `latency_ms`
- `search_result_clicked`
  - params: `query`, `result_type`, `result_id`, `rank`
- `search_no_result_viewed`
  - params: `query`, `search_type`
- `search_retried`
  - params: `prev_query`, `new_query`

## 4. 알림 이벤트
- `notification_received`
  - params: `notification_type`, `target_type`, `target_id`
- `notification_clicked`
  - params: `notification_type`, `target_type`, `target_id`
- `notifications_marked_all_read`
  - params: `unread_count_before`

## 5. 경매툴 100명 퍼널 이벤트
- `auction_tool_landing_viewed`
  - params: `campaign`, `creative_id`, `channel`, `device`
- `auction_created`
  - params: `auction_id`, `category`, `starting_price`, `duration_hours`
- `auction_shared`
  - params: `auction_id`, `share_channel`, `share_target`
- `auction_first_bid`
  - params: `auction_id`, `bid_amount`, `bidder_id`, `is_first_bid`
- `auction_completed`
  - params: `auction_id`, `final_price`, `bid_count`, `winner_id`
- `auction_revisited`
  - params: `auction_id`, `days_since_create`, `entry_point`

## 6. KPI 매핑
- 거래 완료율: `product_status_changed(to_status=판매완료)` 기반 산출
- 검색 무결과 비율: `search_no_result_viewed / search_executed`
- 알림 클릭률: `notification_clicked / notification_received`
- 경매 생성 전환율: `auction_created / auction_tool_landing_viewed`
- 경매 공유 전환율: `auction_shared / auction_created`
- 첫 입찰 전환율: `auction_first_bid / auction_created`
- 경매 완료율: `auction_completed / auction_created`
- 7일 재방문율: `auction_revisited(days_since_create<=7) / auction_created`

## 7. 품질 점검 체크리스트
- 동일 액션에서 이벤트 중복 발화가 없는가
- 핵심 이벤트 누락 화면이 없는가
- 대시보드 지표와 DB 집계 수치가 큰 차이가 없는가
