# 카테고리 사전 v1

작성일: 2026-02-11  
목적: 상품/검색/운영에서 동일한 카테고리 언어를 사용하기 위한 기준표

## 1. 최상위 분류 (`productType`)
- `생물`
- `용품`

## 2. 상품 카테고리 (`category`) 제안

### 생물
- 장수풍뎅이
- 사슴벌레
- 타란튤라
- 전갈
- 기타 절지류
- 파충류
- 어류
- 기타 생물

### 용품
- 사육장
- 바닥재
- 먹이
- 온습도/조명
- 장식/은신처
- 관리도구
- 기타 용품

## 3. 게시글 카테고리 (`post.category`) 제안
- 자유
- 질문
- 정보
- 자랑
- 후기
- 공지

## 4. 상태값 (`product.status`) 운영 기준
- 판매중
- 예약중
- 판매완료

주의:
- 상태값은 한글 기준으로 유지하되 API/프론트에서 동일 문자열만 사용
- 신규 상태 추가 시 운영 정책/필터 UI 동시 업데이트

## 5. 필터 파라미터 규칙
- `category`: 위 사전 값만 허용
- `productType`: `생물 | 용품`
- `status`: `판매중 | 예약중 | 판매완료`
- `전체`는 UI 전용 값으로, API에는 전달하지 않음

## 6. 데이터 정리 원칙
- `null` 카테고리는 우선 `기타 생물` 또는 `기타 용품`으로 매핑
- 자동 매핑이 불가능한 데이터는 운영 검수 큐에 적재
- 기존 데이터 정리 후 신규 등록에서 `null` 입력 차단
