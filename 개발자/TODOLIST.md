# PRD: 브리디 핵심 성장 기능 개편

- 작성일: `2026-02-23`
- 작성자: 기획/제품 관점 초안
- 문서 목적: 브리디의 핵심 가치(경쟁/인정/거래)를 전면에 배치해 참여율과 재방문율을 높인다.

## 1) 문제 정의

1. 현재 홈은 기능이 분산되어 사용자가 첫 화면에서 핵심 행동을 빠르게 이해하기 어렵다.
2. `브리디북`은 보상 지연형 기능이라 대다수 사용자에게 즉각적 동기(재미/경쟁/거래 성과)가 약하다.
3. 커뮤니티, 경매, 프로필이 하나의 성장 루프로 연결되지 않아 참여가 단발성으로 끝난다.
4. 랭킹/신뢰/성과가 시각적으로 약해 서비스의 브랜딩 메시지가 흐려진다.

## 2) 제품 가설

1. 홈 첫 화면에 `인기 브리더`, `인기 혈통`, `카테고리별 최고가 경매`를 전면 배치하면 참여 행동(게시/댓글/입찰)이 증가한다.
2. 명예 요소(시즌 랭킹, 배지, 공유 카드)를 추가하면 UGC 생성과 공유율이 증가한다.
3. 브리디북을 메인 기능이 아닌 증빙 레이어로 재배치하면 기능 이해가 쉬워지고 핵심 퍼널 이탈이 줄어든다.

## 3) 목표/KPI

1. `주간 활성 브리더 수(WAB)` 30% 상승
2. `랭킹 참여율` 25% 상승
3. `경매 조회 -> 입찰 전환율` 20% 상승
4. `홈 첫 클릭의 랭킹/경매 관련 비중` 40% 이상
5. `7일 리텐션` 15% 상승

## 4) 핵심 포지셔닝

`브리디 = 브리더의 실력과 신뢰를 증명하는 무대`

- 코어 기능 3축:
  1. 인기 브리더 랭킹
  2. 인기 혈통 랭킹
  3. 카테고리별 최고가 경매 랭킹
- 브리디북 역할:
  1. 메인 유입 기능 아님
  2. 상위 유저/판매자의 신뢰 증빙 기능

## 5) 범위 정의

### Must (P0)

1. 홈 IA 개편: 랭킹 3축 전면 배치
2. 실시간 인기 브리더 TOP 섹션
3. 인기 혈통 TOP 섹션
4. 카테고리별 최고가 경매 랭킹 섹션
5. 급상승 커뮤니티 글 섹션(짧은 카드형)
6. 시즌 랭킹(주간) + 프로필 배지 기본

### Should (P1)

1. 미션형 참여 장치(게시/댓글/입찰)
2. 랭킹 진입 공유 카드 생성
3. FOMO 알림(순위 하락/추월/최고가 갱신)
4. 개인화 홈 섹션(내 순위/다음 미션)

### Could (P2)

1. 브리디북의 신뢰 점수 연동
2. 운영 리포트 자동 발행(주간 하이라이트)
3. 오프플랫폼 공유 확장(오픈채팅/인스타 스토리 템플릿)

## 6) 홈 정보 구조(모바일 기준)

1. `Hero`: 이번 주 TOP 브리더 + `내 순위 올리기` CTA
2. `랭킹 1`: 카테고리별 최고가 경매 TOP
3. `랭킹 2`: 인기 혈통 TOP
4. `커뮤니티`: 이번 주 급상승 글
5. `개인화`: 내 순위/내 진행 미션

## 7) 기능 요구사항

### 7-1. 인기 브리더 랭킹

1. 노출 항목: 순위, 닉네임, 점수, 최근 변동(상승/하락), 대표 뱃지
2. 정렬 기준: 최근 7일 기여도 가중치(게시/댓글/입찰/낙찰)
3. 상세 진입: 프로필 + 활동 근거

### 7-2. 인기 혈통 랭킹

1. 노출 항목: 혈통명, 관심수, 거래수, 평균가, 최근 상승률
2. 정렬 기준: 관심/거래/반응 복합 점수
3. 상세 진입: 관련 경매/게시글 묶음

### 7-3. 카테고리별 최고가 경매

1. 노출 항목: 카테고리, 최고가, 판매자, 낙찰일
2. 필터: 카테고리/기간(주간, 월간, 누적)
3. 상세 진입: 경매 상세 + 검증 상태

### 7-4. 급상승 커뮤니티

1. 노출 항목: 제목, 요약 2줄, 작성자, 반응수, 상승 배지
2. 정렬 기준: 24시간 반응 증가량
3. 카드 정책: 긴 본문 미노출, 목록 가독성 우선

### 7-5. 시즌/배지

1. 시즌 단위: 주간 기본, 월간 확장
2. 배지 유형: TOP 브리더, 최고가 낙찰, 인기 혈통
3. 배지 위치: 프로필, 경매 카드, 게시글 카드

## 8) 브랜딩/마케팅 연결

1. 메인 카피: `실력은 기록되고, 신뢰는 거래로 증명된다`
2. 주간 하이라이트 카드: `이번 주 TOP 브리더`, `최고가 경매`, `급상승 혈통`
3. 공유 유도: 랭킹 진입 시 1클릭 공유 카드

   - props: `auction_id`, `category`, `current_price`
4. `challenge_join`

5. `share_card_export`
   - props: `card_type`, `channel`

## 10) 출시 계획

1. Phase 1 (2주): 홈 IA 개편 + 랭킹 3축 + 급상승 커뮤니티
2. Phase 2 (2주): 시즌/배지 + 공유 카드 + 개인화 홈
3. Phase 3 (2주): 미션/알림 고도화 + 브리디북 증빙 연동

## 11) 리스크 및 대응

1. 랭킹 조작 시도
   - 대응: 비정상 활동 탐지, 점수 반영 제한
2. 신규 유저 소외
   - 대응: 신규 전용 랭킹/미션 트랙 제공
3. 콘텐츠 품질 저하
   - 대응: 신고/제재 정책 노출 강화, 운영 큐레이션

## 12) 비목표(Non-goals)

1. 브리디북 전면 리브랜딩
2. 전체 IA 전면 개편(핵심 퍼널 외)
3. 외부 결제/정산 정책 변경
