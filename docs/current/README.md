# 현재 기준 문서

이 디렉터리는 현재 제품 구조, 설계 방향, 기술 레퍼런스를 담습니다.

현재 기준의 기술 베이스라인은 다음을 전제로 합니다.
- Next.js 16
- App Router + Pages API Routes 혼용
- `proxy.ts` 기반 라우팅 가드
- `instrumentation-client.ts` / `app/global-error.tsx` 기반 Sentry 설정
- ESLint Flat Config (`eslint.config.mjs`)
