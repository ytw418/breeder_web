# BREDY (브리디) 서비스 개요

## 1. 서비스 정체성

브리디(Bredy)는 희귀 애완생물(파충류, 곤충, 희귀식물 등)을 중심으로 한
글로벌 커뮤니티 기반 플랫폼이다.

브리디는 거래의 당사자나 중개자가 아니며,
이용자 간 정보 공유 및 경매 기능을 제공하는 중립 플랫폼이다.

운영 형태:
- 개인 개발자가 운영
- 대한민국 기반 서비스
- 글로벌 확장 지향

---

## 2. 핵심 목표

1. 희귀생물 대표 커뮤니티 플랫폼
2. 경매 중심의 참여형 생태계 구축
3. 브리더 활동 기반 신뢰 구조 형성
4. 장기적으로 글로벌 니치 플랫폼 1위

---

## 3. 주요 기능

### 1) 커뮤니티 기능
- 게시글 작성
- 댓글
- 프로필 페이지
- 사용자 간 소통

### 2) 경매 시스템
- 회원 간 입찰 기반 거래 기능
- 실시간 입찰 구조
- 낙찰자 확정 시스템

※ 브리디는 거래에 직접 관여하지 않음.

### 3) 혈통 정보 표시 기능
- 이용자가 제출한 정보를 기반으로 한 표시 기능
- 운영자가 진위 여부를 보증하지 않음

### 4) 인증 브리더 표시
- 내부 활동 기준 충족 여부 표시
- 신뢰성, 품질, 적법성 보증 아님

---

## 4. 법적 포지션

브리디는 다음과 같은 입장을 유지한다:

- 중립 플랫폼
- 거래 보증하지 않음
- 생물의 적법성 보증하지 않음
- CITES 및 수출입 책임은 이용자에게 있음
- 서비스는 AS-IS 제공

---

## 5. 수익 모델 (예정 포함)

- 경매 수수료
- 인증 브리더 구독
- 광고 수익
- 프리미엄 노출
- 데이터 기반 확장 모델 (장기)

---

## 6. 타겟 시장

1차 타겟:
- 파충류 브리더 및 매니아

확장 가능 분야:
- 곤충
- 희귀식물
- 글로벌 희귀생물 시장

---

## 7. 톤 & 브랜드 방향

브리디는 다음과 같은 이미지를 지향한다:

- 전문적
- 과장하지 않음
- 중립적
- 플랫폼 중심
- 커뮤니티 기반

사용 금지 표현:
- "공식 인증"
- "완전 보증"
- "안전 거래 보장"
- "브리디가 검증함"

권장 표현:
- "내부 기준 표시"
- "이용자 제출 기반 정보"
- "중립 플랫폼"

---

## 8. AI 활용 가이드

AI에게 요청 시 다음을 참고하도록 한다:

- 브리디는 중립 플랫폼이다.
- 거래 책임은 사용자에게 있다.
- 인증은 보증이 아니다.
- 글로벌 확장을 고려한 톤을 유지한다.
- 법적 책임을 발생시키는 표현은 피한다.

예시 요청:
- "이 파일을 참고해서 공지사항 작성해줘"
- "이 플랫폼 정체성에 맞는 마케팅 카피 작성해줘"
- "경매 이벤트 공지문 작성해줘"

---

## 9. 향후 확장 방향

- 글로벌 영어 서비스 강화
- Stripe/PayPal 연동
- 데이터 기반 시세 분석
- 글로벌 커뮤니티 구축

---

## 10. 운영 철학

브리디는 다음을 지향한다:

- 생태계 중심 플랫폼
- 이용자 주도 성장
- 책임은 당사자 원칙
- 최소 개입, 최대 확장
