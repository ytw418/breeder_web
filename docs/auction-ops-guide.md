# 경매 도구 운영 가이드

작성일: 2026-02-11

## 1) 서비스 포지셔닝 문구

- 이 서비스는 커뮤니티를 대체하는 플랫폼이 아니라,
  카페/밴드/오픈채팅에서 바로 쓰는 경매 진행 도구입니다.
- 링크 생성 -> 공유 -> 입찰 참여를 빠르게 만드는 것이 핵심입니다.

## 2) 판매자 등록 권장 가이드

- 경매 등록 시 다음 정보를 가능한 많이 입력 권장:
  - 블로그/프로필 URL
  - 카페 닉네임
  - 밴드 닉네임
  - 연락 이메일/전화번호
  - 커뮤니티 프로필 캡처 이미지
- 이 정보가 있으면 참여자의 신뢰 확보와 입찰률 상승에 유리합니다.

## 3) 참여자 안내 기본 문구

- 카카오 로그인 기반으로 참여합니다.
- 입찰은 취소할 수 없습니다.
- 마감 임박(3분 이내) 입찰 시 5분 자동 연장됩니다.
- 정책 위반 시 계정 영구 참여 제한이 적용될 수 있습니다.

## 4) 카페/밴드 게시용 템플릿

제목 예시:
- `[외곤경매] 헤라클레스 DHH 150 수컷`

본문 템플릿:
```text
[경매 링크]
https://(서비스도메인)/auctions/(id)

[경매 요약]
- 시작가: 00,000원
- 최소 입찰 단위: 자동 계산
- 종료시각: 2026-02-11 22:00
- 마감 3분 이내 입찰 시 5분 자동 연장

[판매자 신뢰정보]
- 카페 닉네임: OOO
- 밴드 닉네임: OOO
- 블로그/프로필: https://...

[주의]
- 입찰 취소 불가
- 허위/분쟁 유도/부정행위 신고 가능
```

## 5) 신고 처리 운영 기준

- 처리 상태:
  - `OPEN`: 접수
  - `RESOLVED`: 처리 완료
  - `REJECTED`: 기각
- 제재 액션:
  - `NONE`: 경고/모니터링
  - `BAN_USER`: 영구정지

권장 처리 기준:
- 허위매물, 반복 부정행위, 명확한 사기 정황: `BAN_USER`
- 증거 불충분/오해 가능성: `REJECTED` + 재신고 안내
- 경미 위반: `RESOLVED` + 내부 모니터링

## 6) 초기 KPI 운영 기준

- 주간 생성 경매 수
- 첫 입찰 전환율
- 신고율(신고/완료 경매)
- 재사용 셀러 비율

주간 회고 질문:
- 신고가 어떤 구간에서 많이 발생했는가?
- 입찰이 끊기는 구간(로그인/알림/입력)에 병목이 있는가?
- 신뢰정보 입력이 있는 경매의 완주율/낙찰률이 더 높은가?
