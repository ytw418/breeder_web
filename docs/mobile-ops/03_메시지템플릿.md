# 카톡 메시지 템플릿

작성일: 2026-02-12

아래 템플릿을 그대로 복사해서 카톡에 붙여넣으면 된다.

## 1) 데이터 수집 제출 템플릿

```text
#수집
소스: (Yahoo/Naver/Instagram)
카테고리: (곤충/어류/식물/파충류/용품/기타)
제목: 
가격: 
상태: (미사용/사용감/미기재)
지역: 
배송: (직거래/택배/둘다)
링크: 
한줄요약: 
```

## 2) 신고 분류 제출 템플릿

```text
#신고
게시물/상품 ID: 
사유요약: 
분류: (긴급/일반/보류)
권장조치: (숨김/경고/영구정지검토/보류)
근거링크(있으면): 
```

## 3) 공지 초안 제출 템플릿

```text
#공지
제목: 
본문1(변경내용): 
본문2(영향대상): 
본문3(적용시점): 
```

## 4) 하루 마감 보고 템플릿

```text
#완료
이름: 
오늘 처리건수: 
문제 1개: 
내일 우선작업: 
```

## 5) 스킵 보고 템플릿 (접근실패/로그인필요)

```text
#스킵
소스: 
이유: (로그인필요/접근불가/중복/광고성)
링크: 
```
