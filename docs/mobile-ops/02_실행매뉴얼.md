# 실행 매뉴얼 (핸드폰만 사용)

작성일: 2026-02-12

## A. 데이터 수집 담당 매뉴얼

### 목표
- 실거래형 데이터만 수집해서 운영자에게 전달

### 순서
1. 소스 열기
 - Yahoo Auctions
 - 네이버 섹션/카페 검색
 - 인스타 검색(해시태그/키워드)
2. 제목/가격/카테고리/설명/링크 확인
3. `#수집` 템플릿으로 카톡 전송
4. 하루 20건 채우면 `#완료` 전송

### 수집 기준
- 필수: 출처 링크, 제목, 가격(숫자), 카테고리
- 권장: 상태(미사용/사용감), 지역, 배송방식
- 제외: 가격 없는 홍보글, 출처 불명, 중복 링크

## B. 어드민 매니저 보조 매뉴얼

### 목표
- 운영자가 바로 처리할 수 있게 신고/공지/상품 후보를 정리

### 순서
1. 신고 목록 확인
2. 아래 3개로 1차 분류
 - `긴급`: 사기/욕설/반복 도배
 - `일반`: 규정 위반 의심
 - `보류`: 근거 부족
3. 공지 초안 1건 작성
4. 숨김/검토 후보 상품 5건 제출
5. `#완료` 전송

### 공지 초안 규칙
- 3줄 구조 유지
1. 무엇이 바뀌는지
2. 누구에게 영향 있는지
3. 언제부터 적용되는지

## C. 운영자(너) 처리 루틴
1. 카톡 `#수집` 모아 `service-initial-data.json` 반영
2. `#신고` 우선순위대로 어드민 처리
3. `#공지` 초안 교정 후 어드민 등록
4. 하루 1회 결과 공유

## D. 절대 규칙
- JSON/코드 파일 직접 수정 금지
- 출처 없는 데이터 등록 금지
- 개인정보(전화/계좌) 외부 재공유 금지
