# 당신은 관리자로 임명 받았습니다.

작성일: 2026-02-12
대상: 비개발자 운영 담당(핸드폰 사용)

## 해야할일

1. 카카오 로그인하기
- 링크: `/auth/login`
- 카카오로 로그인 후 마이페이지로 이동

2. 카카오 계정 이메일을 성준이에게 전달하기
- 마이페이지에서 내 이메일 확인
- 카톡으로 성준이에게 전달

3. 성준이가 관리자 계정으로 바꿔주면
- 다시 로그인
- 마이페이지에서 `관리자 페이지로 이동` 버튼 확인

4. 관리자 페이지에서 해야 하는 것
- 1순위: `데이터 생성하기` 버튼 누르기
- 2순위: `공지 등록하기` 작성 후 등록

## 매일 업무 (아주 간단)
- 오전: 데이터 생성하기 1회
- 오후: 공지 1개 등록
- 마감: 성준이에게 “오늘 완료” 메시지 보내기

## 문제 생기면
- 버튼이 안 보임: 다시 로그인 후 확인
- 관리자 페이지 안 들어가짐: 성준이에게 이메일 다시 전달
- 오류 메시지 뜸: 화면 캡처해서 성준이에게 전달
