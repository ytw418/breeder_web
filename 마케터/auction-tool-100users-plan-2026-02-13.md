## 4) 퍼널 KPI
- 유입 수: 3,000 (2주 누적)
- 경매 생성 전환율: 10% (300 생성)
- 공유 전환율: 50% (150 공유)
- 첫 입찰 전환율: 40% (120 첫 입찰 도달)
- 7일 재방문율: 35% (생성/입찰 유저 기준)
- CAC: 10,000원 이하 (유료 채널 기준)

### 100명 달성 경로 (숫자 검증)
- 외부+내부 총 유입 3,000명
- 생성 전환 10% = 300명
- 첫 입찰 전환 40% = 120명
- 활성 사용자 정의 기준(생성 또는 첫 입찰 수행) 충족 사용자 100명 이상 확보

## 7) 퍼널 측정 방법 (기획자 검수 대응)
- 퍼널 정의: `유입 -> 생성 -> 공유 -> 첫 입찰 -> 완료 -> 재방문`
- 측정 이벤트(요구 반영):
  - `auction_tool_landing_viewed`
  - `auction_created`
  - `auction_shared`
  - `auction_first_bid`
  - `auction_completed`
  - `auction_revisited`
- 대시보드 산식:
  - 생성 전환율 = `auction_created / auction_tool_landing_viewed`
  - 첫 입찰 전환율 = `auction_first_bid / auction_created`
  - 완료 전환율 = `auction_completed / auction_created`
  - 7일 재방문율 = `auction_revisited(7d) / auction_created`
